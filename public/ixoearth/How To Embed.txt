1. Upload "ixo Earth" folder to your website via FTP
2. Add this custom css:

.intrinsic-container {
  position: relative;
  height: 0;
  overflow: hidden;
}
 
/* 16x9 Aspect Ratio */
.intrinsic-container-16x9 {
  padding-bottom: 56.25%;
}
 
/* 4x3 Aspect Ratio */
.intrinsic-container-4x3 {
  padding-bottom: 75%;
}
 
.intrinsic-container iframe {
  position: absolute;
  top:0;
  left: 0;
  width: 100%;
  height: 100%;
}

3. On the page you want to embed the earth, add this HTML code:

<div class="intrinsic-container intrinsic-container-16x9">
<iframe src = "https:/URL TO WHERE YOU COPIED THE FOLDER/ixo Earth/index.html" 
		allowfullscreen
		scrolling = "no">
</iframe>
  </div>

4. Change the iframe src to wherever you copied the ixo Earth folder.

5. Enjoy!
